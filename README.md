# Air Quality Analysis

## Overview

This project explores Air Quality Index (AQI) data using Python for data analysis and visualization.

The goal is to practice fundamental data science techniques including:

* Data handling with Pandas
* Data visualization with Matplotlib
* Exploratory data analysis
* Environmental data interpretation

## Technologies Used

* Python
* Pandas
* Matplotlib

## Project Structure

Air-Quality-Analysis/

├── air-quality-analysis.py

└── README.md

## Current Features

* Creation of AQI datasets
* DataFrame manipulation
* AQI trend visualization

## Future Improvements

* Build predictive models
